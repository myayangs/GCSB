>In the process of creating a conversational agent, what are the two distinct types of conversational paths you can choose from to get started?
```
Playbook and Flow
```

>What are the three steps for configuring conditional actions for playbooks?
```
Trigger, Condition, Action
```

>Which of the following is considered a best practice when building Generative Playbooks?
```
Clarity: Provide clear and specific instructions and examples to ensure the LLM understands the desired behavior and to enhance the quality of the output.
```


>Which four components are essential for a Generative Playbook?
```
Goal, Steps, Examples, Tools
```

>Which of the following is a correct format used to define the OpenAPI Schema for Playbook Tools?
```
JSON or YAML
```

>What are two ways to create a hybrid agent?
```
By invoking a flow within a playbook, or by invoking a playbook within a flow.
```
